# Manufacturing Operations Analytics Dashboard

## Overview

The Manufacturing Operations Analytics Dashboard is an end-to-end data analytics project designed to monitor and improve manufacturing performance. This project leverages Excel, SQL, Power BI, and Tableau to analyze production data, track operational KPIs, and generate actionable business insights.

## Objectives

* Monitor production performance and efficiency.
* Analyze machine utilization and downtime.
* Track product quality and defect rates.
* Identify operational bottlenecks.
* Support data-driven decision-making through interactive dashboards.

## Technologies Used

* Microsoft Excel
* SQL
* Power BI
* Tableau

## Project Workflow

### 1. Data Collection

Collected manufacturing data containing production metrics, machine performance, downtime records, and quality information.

### 2. Data Cleaning and Preparation

* Removed duplicate records.
* Handled missing values.
* Standardized data formats.
* Prepared datasets for analysis using Excel.

### 3. Data Analysis Using SQL

* Extracted and transformed data.
* Performed KPI calculations.
* Analyzed production trends and operational metrics.

### 4. Dashboard Development in Power BI

Created interactive dashboards featuring:

* Production Volume Analysis
* Machine Utilization Tracking
* Downtime Monitoring
* Quality Performance Metrics
* KPI Reporting

### 5. Data Visualization in Tableau

Developed visual reports and dashboards to:

* Identify production trends.
* Compare departmental performance.
* Analyze operational efficiency.

## Key Performance Indicators (KPIs)

* Total Production
* Machine Utilization Rate
* Downtime Hours
* Defect Rate
* Production Cost
* Overall Equipment Effectiveness (OEE)

## Key Features

* Interactive Dashboards
* Real-Time KPI Monitoring
* Production Trend Analysis
* Downtime Analysis
* Quality Performance Tracking
* Business Intelligence Reporting

## Business Impact

This dashboard helps manufacturing teams identify inefficiencies, reduce downtime, improve production quality, and make informed business decisions based on data-driven insights.
